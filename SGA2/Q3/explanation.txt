System Calls Used

1. open()
Creates or opens the employee records file and returns a file descriptor.

2. write()
Stores employee records into the file.

3. lseek()
Moves the file pointer directly to a specific employee record for updating or reading.

4. read()
Reads a selected employee record from the file.

5. close()
Closes the file and releases system resources.
